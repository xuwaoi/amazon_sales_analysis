# Amazon Sales SQL Analysis

SQL аналитический проект по анализу продаж Amazon.

## Dataset
Amazon Sales dataset (2022).

## Цель анализа

Проанализировать структуру продаж, сезонность, категории товаров и эффективность ассортимента.

## Используемые инструменты

- MySQL
- SQL
- Window Functions
- CTE

## Основные этапы анализа

### Data Quality Check
- проверка NULL значений
- поиск дубликатов заказов

### Revenue Analysis
- динамика выручки по месяцам
- накопительная выручка
- сезонность продаж

### Product Analysis
- доля категорий в выручке
- топ товары в категории
- ABC анализ SKU

### Customer Segmentation
- B2B vs B2C анализ

### Order Status Analysis
- потери выручки из-за отмен и возвратов
- распределение статусов заказов

### Geographic Analysis
- топ городов по выручке

## Основные инсайты

- продажи имеют выраженную сезонность с пиком в апреле–июне
- категории **Set** и **Kurta** формируют основную часть выручки
- основной сегмент клиентов — **B2C**
- небольшая доля товаров формирует большую часть выручки (принцип Pareto)

## SQL техники

В проекте используются:

- CTE
- Window Functions
- Ranking
- Running Totals
- ABC Analysis
